# Lista de Tarefas - Jogo de Xadrez Godot

Esta é a lista de funcionalidades a serem implementadas para finalizar o jogo, em ordem de desenvolvimento sugerida.

## Próximas Funcionalidades (Ordem Sugerida)

- [x] **Indicador de Turno:** Adicionar um elemento visual que mostre de qual cor (branca/preta) é a vez de jogar.
- [ ] **Efeitos Sonoros (Sound Effects):** Adicionar sons para as ações do jogo.
    - [ ] Som ao mover uma peça.
    - [ ] Som ao capturar uma peça.
    - [ ] Som ao entrar em xeque ou xeque-mate.
- [ ] **Promoção de Peão:** Implementar a lógica e a interface para permitir que o jogador escolha uma nova peça quando um peão alcança o final do tabuleiro.
- [ ] **Roque (Castling):** Implementar o movimento especial do Rei e da Torre, incluindo as condições necessárias (peças não movidas, caminho livre, não estar em xeque, etc.).
- [ ] **En Passant:** Implementar a regra de captura especial do peão.

## Melhorias de Interface e UX

- [ ] **Botão "Novo Jogo":** Adicionar um botão na tela para reiniciar a partida a qualquer momento.
- [ ] **Exibição de Peças Capturadas:** Criar áreas na interface para mostrar as peças que cada jogador capturou.

## Regras Avançadas (Opcional - a discutir)

- [ ] **Empate por Repetição:** Detectar se a mesma posição do tabuleiro se repetiu 3 vezes.
- [ ] **Empate por Regra dos 50 Movimentos:** Detectar se 50 movimentos foram feitos sem a movimentação de um peão ou uma captura.
